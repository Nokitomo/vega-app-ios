# Documentazione Vega

Questa cartella raccoglie la documentazione tecnica e funzionale del progetto.
Le sezioni includono note su localizzazione (i18n) e supporto lingua inglese/italiano.

## Indice
- overview.md: panoramica dell'app e delle funzionalita principali.
- architecture.md: struttura del progetto, entrypoint e moduli principali.
- navigation.md: stack di navigazione e rotte.
- state-and-storage.md: stato globale (Zustand) e persistenza (MMKV).
- providers-and-extensions.md: sistema provider dinamici e moduli esterni.
- services.md: servizi applicativi (download, notifiche, aggiornamenti).
- build-and-run.md: setup, build e avvio in locale.
- testing-and-quality.md: linting e test.
- configuration.md: configurazioni Expo/Metro/Tailwind/NativeWind.
- permissions.md: permessi Android usati e motivazioni.
- troubleshooting.md: problemi noti e risoluzioni (incluso splash bloccato).
